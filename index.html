About This Platform

This tutor marketplace connects students with qualified tutors for 1‑on‑1 and group sessions. Students can discover tutors by subject, availability, and rating, while tutors manage their schedules, pricing, and sessions in one place.

For Students

  • Browse tutors by subject, level, and price.
  • Book sessions based on real‑time availability.
  • Track upcoming sessions and past learning history.

For Tutors

  • Create a detailed profile with subjects, bio, and hourly rate.
  • Manage availability, bookings, and session history.
  • View earnings and payout summaries.

Admin Panel

The admin side of the system provides real‑time views of users and tutors, including:

  • Live tutor list with status (active / suspended).
  • Ability to add or update tutors and see changes instantly.
  • Monitoring of overall platform health and usage.

The admin dashboard is powered by a React view (e.g. TutorAdminView.jsx) that listens for real‑time updates from the backend, so new tutors appear without refreshing.
